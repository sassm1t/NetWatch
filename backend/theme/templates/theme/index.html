A Network Traffic Analysis Tool

NetWatch is a network protocol analyzer.
It lets you capture and interactively browse the traffic running on a computer network.


Get started!





NetWatch is a comprehensive solution combining packet sniffing and network mapping to identify potential vulnerabilities
and threats in large-scale computer networks, leveraging open-source tools and libraries to create a user-friendly
and accessible web-based platform for network administrators.
